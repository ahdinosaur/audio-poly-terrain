# audio-poly-terrain

3d polygon terrain of [frequency, time, value]

## how

### install

```
git clone https://github.com/ahdinosaur/audio-poly-terrain
cd audio-poly-terrain
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
